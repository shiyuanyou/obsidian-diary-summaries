# 配置说明

## data.json 配置文件

插件使用 `data.json` 文件存储配置信息，包括：

### AI服务配置
- OpenAI API 密钥和连接信息
- Ollama 服务配置
- 多个AI服务连接管理

### 汇总类型配置
- 周报、月报、季报、年报的配置
- 每种类型的系统提示词、token限制等

### 日记配置
- 日记文件的根目录路径
- 输出目录配置

## 使用方法

1. 复制 `data.example.json` 为 `data.json`
2. 在插件设置界面中配置你的API密钥和其他参数
3. 或者手动编辑 `data.json` 文件（API密钥会被自动加密存储）

## 注意事项

- `data.json` 文件包含敏感信息，已被添加到 `.gitignore` 中
- API密钥在存储时会被自动加密
- 请勿将包含真实API密钥的配置文件提交到版本控制系统中
